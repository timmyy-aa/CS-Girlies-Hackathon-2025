What Went Into the Creation of Buddy

The creation of Buddy involved a combination of user-focused design, intentional problem-solving, and modern development tools. I built it specifically for college students to help curb procrastination and encourage mutual support.

The entire user interface was first designed in Figma, where I focused on simple layouts, intuitive navigation, and features tailored to student workflows. After finalizing the design, I implemented the application using code written and refined with the help of large language models such as Claude and ChatGPT. This allowed me to quickly prototype features, iterate on ideas, and bring the full concept to life efficiently.

Overall, Buddy blends thoughtful UI/UX design with AI-assisted development to create a practical, motivating, and supportive productivity tool for students.
